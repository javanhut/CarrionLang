## Carrion Lang

# Dynamically Typed, Interpreted, Object Oriented Programming Language Built in GO.


# This is a personal project to learn how interpreters work and i'm personally a big fan of python. I wanted to make a programming Language with the ease of use of python and the concurrency of GO.

# I'm planning on just making this as a personal project for fun but if it becomes useful as some point i'll keep working on it's features to make it a robust language.


# For now, it's just gonna be a programming language named after my favorite type of bird, the Crow. Geting it's name from the Carrion Crow. Caledon after the Caledonian Crow was already taken.

# The syntax is inspired by python and crows good luck understanding it.
